Options must be separated by a space.  Any non-ambiguous initial prefix of a
valid option may be used as an abbreviation for that option.  When an option is
composed of two separate words, an abbreviation may be formed by concatenating
the first character of each word.  Options that are switches may be negated by
prefixing them with the string "no".  Default input file extension is ".g"
