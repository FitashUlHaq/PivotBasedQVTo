1) String literal "" should be supported (currently only '' is allowed)

2) self.attribute->object(a) Column {…}" as a shorthand for "self. attribute->xcollect(a| object Column {…})
	is not supported

3) Meta-model definition embedded in a compilation unit is not supported

4) No implicit meta-model resolution by name, explicit modeltype definition is required

5) isKindOf(), etc. aliases not supported, migrated to oclIsKindOf()

6) The 'Encapsulation' example has been adjusted to omg.uml2 metamodel
	to leverage existing models as input

7) merge, inherits do not support referencing by local name (qualified only)

8) Composite assignments are not supported
constraint := {
	self.constraint[isStereotypedBy("precondition")]->map createPrecondition();
	self.constraint[isStereotypedBy("goal")]->map createGoal();
};

9) supplier->resolveone(WorkDefinition) not supported, migrated as 
	supplier.resolveone(WorkDefinition)
